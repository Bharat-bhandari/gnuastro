Gnuastro features that are coming soon (in version 0.15)

GNU Astronomy Utilities (Gnuastro) is an official GNU package consisting of various programs and library functions for the manipulation and analysis of astronomical data.

The current stable Gnuastro release is version 0.14. In this page, you can some demonstrations of ongoing work for the next release. You can always download the tarball containing the most recent work (which includes the features shown here) from this URL: http://akhlaghi.org/gnuastro-latest.tar.lz. We do encourage using this most recent tarball and testing the features to report any possible bugs that have not yet been found.
